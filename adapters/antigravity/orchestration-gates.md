# Antigravity Orchestration Gates

When a request contains a `PACE-[0-9]+` issue key, load and follow:

`skills/pacebuild-orchestrator/SKILL.md`

The wrapper loads the repository-root `PACEBUILD_ORCHESTRATOR.md`. Its state
machine and exact approval tokens override generic instructions to continue
autonomously.

Phase 0 and Phase 1 are read-only. Planning artifacts do not grant permission
to implement. Never infer approval from task clarity, Jira acceptance criteria,
or a previous message.

For model and independent-review behavior, also follow `rules/model-routing.md`.
